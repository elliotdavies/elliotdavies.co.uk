# elliotdavies.co.uk

This is the code for my personal website and blog at [elliotdavies.co.uk](http://elliotdavies.co.uk).

## Working on this project

Install dependencies:

```
yarn
```


Run the site locally:

```
yarn watch
```

Build the production version of the site:

```
yarn build
```
